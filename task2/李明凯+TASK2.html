一、RSI 相对强弱指标

1.1 定义

RSI（Relative Strength Index）是由威尔斯·怀尔德（Welles Wilder）于1978年提出的技术分析指标，用于衡量股价变动的速度和幅度。

1.2 计算公式

RSI = 100 - [100 / (1 + RS)]

其中：

RS = 平均收益 / 平均损失

平均收益 = N日内上涨幅度的平均值

平均损失 = N日内下跌幅度的平均值

常用周期：N = 14日

1.3 判断标准

超买区域：RSI > 70，表示股价可能回调

超卖区域：RSI < 30，表示股价可能反弹

强势区域：RSI > 50，多头市场

弱势区域：RSI < 50，空头市场

背离信号：

• 顶背离：股价创新高，RSI未创新高 → 可能下跌

• 底背离：股价创新低，RSI未创新低 → 可能上涨

二、MACD 指标

2.1 定义

MACD（Moving Average Convergence Divergence）是由杰拉德·阿佩尔（Gerald Appel）于1970年代提出，用于捕捉趋势反转和动量变化的指标。

2.2 计算公式

DIF = EMA(close, 12) - EMA(close, 26)

DEA = EMA(DIF, 9)

MACD = (DIF - DEA) × 2

其中：

EMA = 指数移动平均（Exponential Moving Average）

2.3 判断标准

金叉：DIF由下向上穿越DEA → 买入信号

死叉：DIF由上向下穿越DEA → 卖出信号

多头排列：DIF > DEA > 0 → 强势上涨

空头排列：DIF < DEA < 0 → 强势下跌

柱状图解读：

• 红柱（MACD > 0）：多头力量强

• 绿柱（MACD < 0）：空头力量强

• 柱状体缩短：趋势可能反转

三、布林带（Bollinger Bands）

3.1 定义

布林带是由约翰·布林格（John Bollinger）于1980年代提出，用于衡量价格波动区间和波动率的指标。

3.2 计算公式

中轨 = MA(close, N) （N日移动平均，通常N=20）

上轨 = 中轨 + (K × STD(close, N)) （K通常=2）

下轨 = 中轨 - (K × STD(close, N))

带宽 = 上轨 - 下轨

3.3 判断标准

价格触及上轨：可能回调（超买）

价格触及下轨：可能反弹（超卖）

布林带收窄：波动率降低，即将选择方向

布林带扩张：波动率增加，趋势明确

突破信号：

• 价格突破上轨 → 可能继续上涨（强势突破）

• 价格跌破下轨 → 可能继续下跌（强势跌破）

四、KDJ 随机指标

4.1 定义

KDJ指标是由乔治·莱恩（George Lane）提出，用于捕捉短期超买超卖信号和趋势反转的动量指标。

4.2 计算公式

RSV = (C - L9) / (H9 - L9) × 100

K = EMA(RSV, 3)

D = EMA(K, 3)

J = 3K - 2D

其中：

C = 当日收盘价

L9 = 最近9日最低价

H9 = 最近9日最高价

4.3 判断标准

超买区域：J > 100 或 K/D > 80

超卖区域：J < 0 或 K/D < 20

金叉：K由下向上穿越D → 买入信号

死叉：K由上向下穿越D → 卖出信号

注意事项：

• J值可以为负数或大于100，表示极端行情

• KDJ适合短期交易，MACD适合中长期趋势

• 在强势上涨中，KDJ可能长期停留超买区

五、综合研判

5.1 多指标共振

强势买入信号：

• RSI从超卖区上穿50

• MACD金叉且柱状图由绿转红

• 股价从布林带下轨反弹

• KDJ金叉且J值从负数回升

强势卖出信号：

• RSI从超买区下穿50

• MACD死叉且柱状图由红转绿

• 股价从布林带上轨回落

• KDJ死叉且J值从高位回落

5.2 风险提示

技术指标仅供参考，不构成投资建议。实际交易需结合基本面、市场情绪、资金流向等多方面因素综合判断。
