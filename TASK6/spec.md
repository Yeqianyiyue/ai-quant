## 一、项目概述

本任务以 A 股季度财务因子数据 `model_data.csv` 为载体，构建一个**基于机器学习的量化交易策略**完整流程：

1. **理论讲解**：机器学习交易策略的核心理念、优缺点；

## 二、理论基础（写入 Notebook）

### 第 1 问：基于机器学习模型的交易策略核心理念

#### 1.1 核心理念
- **从数据中学习模式**：传统量化策略（如双均线）依赖人工规则；ML 策略让算法从历史数据中自动发现因子与未来收益之间的非线性关系。
- **预测驱动决策**：ML 模型输出对资产未来走势的预测（分类：涨/跌；回归：收益幅度），将预测信号映射为交易指令。
- **因子组合优化**：ML 天然适合处理高维因子空间，能自动捕捉因子间的交互效应（如估值 × 成长性 × 动量）。

#### 1.2 优点
| 优点 | 说明 |
|------|------|
| 自动化模式发现 | ML 能发现人类难以察觉的复杂非线性关系 |
| 处理高维因子 | 同时处理数十到数百个因子，无惧维度灾难（树模型、正则化） |
| 动态适应 | 在线学习或滚动训练可使模型适应市场风格切换 |
| 可扩展性强 | 同一套框架可迁移到不同资产、市场、周期 |
| 减少主观偏差 | 用数据驱动替代经验直觉，避免情绪化决策 |

#### 1.3 缺点
| 缺点 | 说明 |
|------|------|
| 过拟合风险极高 | 金融数据信噪比极低（高噪声），ML 模型极易学到噪声而非信号 |
| 未来函数 / 数据泄漏 | 特征构造或划分不当会引入未来信息，导致回测"虚假繁荣" |
| 非平稳性 | 市场统计特性随时间变化，历史模式可能失效（概念漂移） |
| 交易成本敏感 | ML 高频换手带来高交易成本，收益被摩擦成本侵蚀 |
| 可解释性差 | 复杂模型（RF/GBM）的决策过程不透明，难以归因和风控 |
| 小样本问题 | 金融数据通常只有数十年，远低于 CV/NLP 的数据量级 |

### 第 2 问：量化交易 ML 模型中常见自变量与应变量

#### 2.1 常见自变量（因子 / Features）

按数据来源分以下几大类：

**基本面因子（本任务主力）**
| 因子 | 含义 | 逻辑 |
|------|------|------|
| PE(TTM) | 市盈率（滚动12个月） | 估值高低 |
| PB(MRQ) | 市净率（最新报告期） | 账面价值溢价 |
| PS(TTM) | 市销率 | 收入估值 |
| 股息率 | 近12个月股息/股价 | 分红回报 |
| MV | 总市值（对数或原始） | 规模因子 |
| 净利润同比增长率 | 盈利增速 | 成长性 |
| 营业收入同比增长率 | 收入增速 | 成长性 |
| ROE 相关（派生） | 净利润/净资产 | 盈利能力 |
| 企业倍数 EV/EBITDA | 企业价值倍数 | 综合估值 |

**技术面因子（可自行补充）**
- 动量因子：N 日收益率、N 日均线偏离度
- 波动因子：N 日波动率、ATR
- 量价因子：成交量变化率、量价背离
- 技术指标：RSI、MACD、布林带

**宏观因子（可引入）**
- 利率、M2 增速、CPI、PMI

#### 2.2 应变量（标签 / Target）

| 类型 | 定义 | 适用场景 |
|------|------|----------|
| **分类（本任务主推）** | `y_t = 1` 若 `Next_Ret > 0`，否则 `0` | 方向性交易（择时） |
| **回归** | `y_t = Next_Ret` 连续值 | 收益幅度预测，更精细但更难 |
| **排序（Rank IC）** | 截面排序收益 | 多空组合、选股 |
| **分箱/多分类** | 按分位数将 `Next_Ret` 分为 3~5 类 | 多档位决策（强买/弱买/观望/卖出） |

**本任务采用**：二分类 `y = (Next_Ret > 0)` 作为主标签。

### 第七部分：基于模型的交易策略（对应第 3 问第 4 步）
- **7.1 策略逻辑**：用测试集预测信号构建多头择时策略
  - `signal_t = 1` 若模型预测 `Next_Ret > 0`（即 `y_prob > 0.5`），否则 `0`
  - 持仓 `position_t = signal_t`（不考虑做空）
- **7.2 季度收益率计算**：按季度（Date）分组，对测试集内每季度分别计算：
  - 策略季度收益 = `position_t * Next_Ret_t` 的季度和（或季度均值）
  - 买入持有季度收益 = `Next_Ret_t` 的季度和
- **7.3 以季度为单位的对比表**：每季度策略 vs 买入持有收益率，标注胜负

### 第八部分：回测与核心指标（对应第 3 问第 5 步）
- **8.1 策略净值计算**：
  - 策略净值：`NAV_strat_t = ∏(1 + position_{t-1} * Next_Ret_t)`，`NAV_0 = 1`
  - 基准净值：`NAV_bench_t = ∏(1 + Next_Ret_t)`
- **8.2 核心指标计算**：
  | 指标 | 公式（沿用 task3） | 说明 |
  |------|---------------------|------|
  | 累计回报 | `NAV_end − 1` | 总收益 |
  | 年化收益率 | `NAV_end^(N_years) − 1` | 年化收益（每年约 4 季度） |
  | 最大回撤 MDD | `min(NAV/Peak − 1)` | 最坏亏损 |
  | 夏普比率（年化） | `mean(r)/std(r) * sqrt(periods_per_year)` | 风险调整后收益 |
  | 胜率 | `sum(signal==1 and Next_Ret>0) / sum(signal==1)` | 预测正确的占比 |
  | 盈亏比 | `avg(positive_ret) / |avg(negative_ret)|` | 赚比亏的倍数 |
- **8.3 多模型指标对比表**：将所有模型的回测指标汇总为一个 DataFrame
