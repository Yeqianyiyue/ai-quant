## 二、理论基础（写入 Notebook 第一/二部分）

### 1. 海龟策略核心思想
- **趋势跟踪（Trend Following）**：不预测顶底，只在价格**突破前期高低点**时顺势入场，赚"已发生趋势延续"的钱。
- **两套系统并行**（Dennis 原版）：
  - **系统一（S1）**：入场 = 20 日高低点通道突破；出场 = 10 日反向突破。
  - **系统二（S2）**：入场 = 55 日高低点通道突破；出场 = 20 日反向突破。
  - 本报告以 S1（20/10）为**默认**，并允许自由调节。
- **波动性头寸管理**：用 ATR 衡量波动，波动越大单笔头寸越小（`单位 = 1% 账户权益 / ATR`），使每笔交易承受的"波动风险"大致相等。
- **金字塔加仓**：入场后价格每向有利方向移动 0.5×ATR，加一个单位（最多 4 单位），让盈利的头寸"滚雪球"。
- **严格止损**：单笔亏损上限 = 2×ATR（价格反向触及即离场），先求"活下来"再求盈利。

### 2. 关键优势（写入 Notebook 二）
| 优势 | 说明 |
|------|------|
| 规则化、可复制 | 完全机械化，无主观判断，避免情绪化交易 |
| 顺势而为 | 大趋势中能持住仓位，吃满主升/主跌段 |
| 波动自适应仓位 | ATR 头寸管理让风险敞口随市况自动缩放 |
| 风险前置 | 2×ATR 硬止损把单笔最大亏损锁死 |
| 正偏分布友好 | 少数大赢覆盖多数小亏，靠"截断亏损、让利润奔跑"盈利 |

### 3. 核心概念解释（写入 Notebook 二，逐条配公式）
- **高低点通道（Donchian Channel）**：
  - 上轨 `Upper_t = max(High_{t-n+1..t})`（n 日最高价）
  - 下轨 `Lower_t = min(Low_{t-n+1..t})`（n 日最低价）
  - 突破上轨 → 做多信号；跌破下轨 → 做空/平多信号。
- **平均真实波幅 ATR（Average True Range）**：
  - 真实波幅 `TR_t = max(H_t−L_t, |H_t−C_{t-1}|, |L_t−C_{t-1}|)`
  - `ATR_t = mean(TR_{t-m+1..t})`（常用 m=14，亦可用 Wilder 平滑）
  - 衡量"一天通常波动多少"，是头寸规模与止损距离的标尺。
- **止损条件**：
  - 多头止损价 `StopLong = Entry − stop_mult × ATR`（默认 stop_mult=2）；
  - 空头止损价 `StopShort = Entry + stop_mult × ATR`；
  - 价格触及止损价即离场，控制单笔最大亏损。

## 五、指标公式速查（写入 Notebook 第五部分）

设策略日收益序列 `r_t`，交易日数 `n`，无风险日收益 `rf_d = rf_annual/252`，策略净值 `NAV_t`，持仓 `pos_t ∈ {−1,0,+1}`。

1. **累计回报** `Cumulative = NAV_n − 1`
2. **年化收益率** `Annualized = NAV_n^(252/n) − 1`
3. **最大回撤 MDD** `MDD = max_t (Peak_t − NAV_t)/Peak_t`，`Peak_t = max(NAV_0..t)`
4. **夏普比率** `Sharpe = (mean(r_t) − rf_d)/std(r_t) × √252`（ddof=1；rf 默认 2.5%）
5. **Beta** `β = Cov(r_strat, r_market)/Var(r_market)`，基准 = 该股票买入持有
6. **特雷诺比率** `Treynor = (R_ann − R_f)/β`
7. **IC / RankIC**（单只股票时间序列适配）：因子取**通道突破强度** `break_t = close_t − upper_t`（或持仓信号 `pos_t`），未来收益取 `r_{t+1}`；
   - `IC = Pearson(break, r_{t+1})`
   - `RankIC = Spearman(break, r_{t+1})`
8. **ICIR** 滚动 60 日窗口一串 IC_t，取 `mean/std` 衡量稳定性
9. **交易次数** `N = Σ|pos_t − pos_{t-1}|`；**胜率** = 已平仓交易中净收益为正占比
