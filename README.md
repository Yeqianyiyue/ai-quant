# AI Quant - 中芯国际 K线可视化

基于 Tushare 数据的中芯国际（688981.SH）近一年行情可视化面板。

## 在线预览

访问：https://yeqianyiyue.github.io/ai-quant/

## 功能

- 📈 K线蜡烛图（中国习惯：红涨绿跌）
- 📊 交易量柱状图（颜色与K线联动）
- 📅 底部滑块缩放查看任意时间段
- 🖱️ 鼠标悬停显示每日详细行情
- 📊 顶部统计卡片（期间最高/最低价、平均成交量、上涨/下跌天数）

## 数据来源

Tushare Pro API — 中芯国际（688981.SH）近一年日线数据

## 本地运行

直接用浏览器打开 `index.html` 即可（数据已内嵌，无需服务器）。
