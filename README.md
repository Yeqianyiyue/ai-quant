# AI Quant - 中际旭创股票分析面板

基于真实交易数据的 A 股个股深度分析面板，由 AI 自动生成。

## 在线预览

访问：https://yeqianyiyue.github.io/ai-quant/

## 功能概览

- 📈 **K线蜡烛图**：展示开盘/收盘/最高/最低价，中国习惯配色（红涨绿跌）
- 📊 **均线系统**：MA5 / MA20 / MA60 三条均线叠加显示
- 📉 **成交量统计**：对应每交易日成交量，支持缩放联动
- 🔀 **时间周期切换**：近一月 / 近三月 / 近六月 / 近一年
- 🏢 **公司概览**：主营业务、行业地位
- 💰 **核心财务数据**：营收、净利润、毛利率、ROE 等关键指标
- 📝 **投资分析**：投资逻辑、驱动因素、核心风险、操作策略
- 📚 **扩展阅读**：量化交易介绍、K线/基本面/技术面概念解析

## 数据说明

- 标的：中际旭创 300308.SZ（深交所创业板）
- 数据范围：2025-07-04 ~ 2026-07-03（共 242 个交易日）
- 数据来源：Tushare Pro

## 本地运行

直接用浏览器打开 `index.html` 即可，数据已内嵌，无需服务器。
